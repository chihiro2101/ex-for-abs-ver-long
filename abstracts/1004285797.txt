mpeg-7 is a multimedia content description standard . it was standardized in iso/iec 15938 ( multimedia content description interface ) . this description will be associated with the content itself , to allow fast and efficient searching for material that is of interest to the user . mpeg-7 is formally called multimedia content description interface . thus , it is not a standard which deals with the actual encoding of moving pictures and audio , like mpeg-1 , mpeg-2 and mpeg-4 . it uses xml to store metadata , and can be attached to timecode in order to tag particular events , or synchronise lyrics to a song , for example . it was designed to standardize : a set of description schemes ( '' ds '' ) and descriptors ( '' d '' ) a language to specify these schemes , called the description definition language ( '' ddl '' ) a scheme for coding the description the combination of mpeg-4 and mpeg-7 has been sometimes referred to as mpeg-47 .
