the conversational monitor system ( cms & ndash ; originally : '' cambridge monitor system '' ) is a simple interactive single-user operating system . cms was originally developed as part of ibm 's cp/cms operating system , which went into production use in 1967 . cms is part of ibm 's vm family , which runs on ibm mainframe computers . vm was first announced in 1972 , and is still in use today as z/vm . cms runs as a '' guest '' operating system in a private virtual machine created by the vm control program . the control program plus cms together create a multi-user time-sharing operating system .
