image : mc68451 p1160081.jpg thumb this 68451 mmu could be used with the motorola 68010 300x300px a memory management unit ( mmu ) , sometimes called paged memory management unit ( pmmu ) , is a computer hardware unit having all memory references passed through itself , primarily performing the translation of virtual address to physical addresses . an mmu effectively performs virtual memory management , handling at the same time memory protection , cache control , bus arbitration and , in simpler computer architectures ( especially 8-bit systems ) , bank switching .
